# Cortex Frontend

### Pre-requisites
- Hugo > v0.68
- Go > v1.11

### Build Locally
    clone repo 
    cd repo
    git submodule update --init --recursive
    hugo serve

After the Hugo server starts, you should be able to access your site at http://localhost:1313

### Build with Docker
    clone repo
    cd repo
    git submodule update --init --recursive
    ./run.sh
